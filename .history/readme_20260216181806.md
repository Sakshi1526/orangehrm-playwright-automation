OrangeHRM Playwright Automation Project

This project is built using Playwright (JavaScript) to automate the OrangeHRM demo application.

Application URL:
https://opensource-demo.orangehrmlive.com

Project Structure:

src

fixtures (testData.js)

pages (LoginPage.js, DashboardPage.js, EmployeePage.js)

tests (login.spec.js, dashboard.spec.js, employee.spec.js)

utils (constants.js, helpers.js)

playwright.config.js

Features:

Login automation (valid and invalid scenarios)

Dashboard navigation testing

Employee module automation

Page Object Model (POM) framework

Separate test data management

Screenshot and video on failure

Trace viewer support

Headed and slow execution support

How to Run:
Install dependencies using npm install.
Install browsers using npx playwright install.
Run tests using npx playwright test.
Open report using npx playwright show-report.
